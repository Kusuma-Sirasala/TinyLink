# TinyLink - URL Shortener

## Description
TinyLink is a full-stack URL shortener built using Node.js, Express, HTML, CSS, and JavaScript.  
It allows users to paste a long URL and generate a short link that redirects to the original URL.

## Features
- Generate short links for any URL
- Redirect short link to the original URL
- Responsive and user-friendly interface
